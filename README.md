# Pomodoro 番茄鐘
[Demo 網址](https://pomodoro-zhong666.now.sh/)

## 使用技術
- TypeScript
- Vue
- Vue router
- Vuex
- Scss


## 專案開發
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
